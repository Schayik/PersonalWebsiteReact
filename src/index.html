Crystal Conflict

This game was originally based on a Unity tutorial named Roguelike. However, almost every aspect is adjusted to make it suitable for this game. The most significant change is the hexagonal board, for which I used my crystallographic knowledge explained under Chemical Influence.
This is why the game is called Crystal Conflict and why I added the collectible / harvestable crystals. Furthermore, I am currently working on a puzzle RPG game which will be available soon.

Chemical Influence

I wanted to work with a hexagonal grid, because it increases the possibilities of the design. In order to do this, I used my crystallographic knowledge to design the grid system. Some atoms / molecules in crystals also orientate in this hexagonal pattern. So instead of using a 90° corner between the axis, I used 120°. (60° would have been a valid choice as well, but chemists use 120° by convention.)
